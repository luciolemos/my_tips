Linux Mint

O Linux Mint é uma distribuição Linux irlandesa, baseada no Ubuntu 18.04.1 LTS (Bionic Beaver) e é executado no ambiente de desktop do Cinnamon 4.0. Vem com o linux-firmware 1.173.2 e o kernel do Linux 4.15.0-20 e inclui mudanças que supostamente tornam o login no desktop mais fácil e amigável. Possui duas versões: uma baseada em Ubuntu (com o qual é totalmente compatível e partilha os mesmos repositórios) e outra versão baseada em Debian. Suporta muitos idiomas, incluindo a língua portuguesa, e utiliza o Cinnamon como seu principal ambiente de desktop. Linux Mint diferencia-se do Ubuntu e do Debian por incluir drivers e codecs proprietários por padrão e por alguns recursos que permitem fazer em modo gráfico(GUI) configurações que em ambos os sistemas são feitas através do modo texto. Utiliza por padrão o desktop Cinnamon, um derivado do GNOME, com um menu no painel inferior junto à barra de tarefas (o MintMenu), similar ao menu do KDE, ou o menu "Iniciar" do Windows. É um sistema operacional moderno, elegante e confortável, que é poderoso e fácil de usar" e possui suporte multimídia pronto para o uso, incluindo alguns softwares proprietários e vem com uma variedade de aplicativos gratuitos e de código aberto.

XAMPP

Instalando o XAMPP:

my tips »

1. Baixando dos repositórios os pacotes:
luciolemos@dell:~$ wget "https://ufpr.dl.sourceforge.net/project/xampp/XAMPP%20Linux/7.3.10/xampp-linux-x64-7.3.10-0-installer.run" -O xampp-installer.run

2. Dando permissão de escria no doretório: luciolemos@dell:~$ chmod +x xampp-installer.run

3. Iniciando o instalador: luciolemos@dell:~$ sudo ./xampp-installer.run

4. Iniciando o gerenciador do XAMPP no modo gráfico: luciolemos@dell:~$ sudo /opt/lampp/manager-linux-x64.run

5. Para iniciar todos os serviços do XAMPP, use o seguinte comando no terminal: luciolemos@dell:~$ sudo /opt/lampp/lampp start

6. Para finalizar todos os serviços do XAMPP: luciolemos@dell:~$ sudo /opt/lampp/lampp stop

7. Reinicializando todod os serviços do XAMPP: luciolemos@dell:~$ sudo /opt/lampp/lampp restart

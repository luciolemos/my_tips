Linux Mint

O Linux Mint é um sistema operacional moderno, elegante e confortável, poderoso e fácil de usar. É uma das distribuições Linux de desktop mais populares do mundo e usada por milhões de pessoas.

Lúcio Lemos Postado em Dezembro 13, 2019

Sua última versão estável, o Linux Mint 19.2, codinome "Tina" é uma versão de suporte de longo prazo LTS (Long Term Support), suportada até o ano de 2023. Ele vem com software atualizado, traz aprimoramentos e muitos novos recursos para tornar sua experiência com a área de trabalho mais confortável. Suas principais caracteŕistcas são:

  • Funciona imediatamente, com suporte completo a multimídia e é extremamente fácil de usar;
  • É gratuito e de código aberto;
  • É orientado pela comunidade;
  • Baseado no Debian e Ubuntu, fornece cerca de 30.000 pacotes e um dos melhores gerenciadores de software;
  • É seguro e confiável; e
  • Suporta muitos idiomas, inclusive o português e utiliza o Cinnamon como seu principal ambiente de desktop.
  • ...

Graças a uma abordagem conservadora de atualizações de software, um Update Manager exclusivo e a robustez de sua arquitetura, o Linux Mint requer pouca manutenção (sem regressões, antivírus, anti-spyware ... etc). Diferencia-se do Ubuntu e do Debian por incluir drivers e codecs proprietários por default e por alguns recursos que lhe permitem fazer em modo gráfico(GUI), configurações que em outros sistemas são feitas através de linha de comando (CLI). Utiliza por padrão o desktop Cinnamon, um derivado do GNOME, com um menu no painel inferior junto à barra de tarefas (o MintMenu), similar ao menu do KDE, ou ao menu "Iniciar" do Windows.

Visite a página oficial do Linux Mint na web, consulte a documentação, baixe o pacote, instale e contribua com o projeto: http://linuxmint.com.

Apache Netbeans

Ambiente de Desenvolvimento Integrado, Plataforma de Ferramentas e Estrutura de Aplicação.

Lúcio Lemos Postado em Dezembro 13, 2019

Sua última versão, o Apache NetBeans 11.2, pode ser instalado em todos os sistemas operacionais compatíveis com Java, como Windows, Linux, Mac OSX e BSD. O Write Once, Run Anywhere, também se aplica ao NetBeans. Com seu Editor Java em constante aprimoramento, muitas funcionalidades avançadas e uma extensa linha de ferramentas, modelos e exemplos, o NetBeans IDE define o padrão de desenvolvimento com suas tecnologias inovadoras.
Algumas de suas característcas são:

  • É gratuito e de código aberto
  • Realça o código-fonte sintática e semânticamente;
  • Permite refatorar facilmente seu código;
  • Incorpora uma variedade de ferramentas úteis e poderosas;
  • O Editor do NetBeans recua linhas, associa palavras e colchetes;
  • Fornece modelos de código, dicas de codificação e ferramentas de refatoração;
  • Suporta várias linguagens, incluindo Java, C/C++, XML, HTML, PHP, Groovy, Javadoc, JavaScript e JSP;
  • Editor extensível, para você poder adicionar suporte para muitas outras linguagens; e
  • Integração inovadora com ferramentas de versionamento como o Subversion, o Mercurial e o Git;
  • ...

É mantido pela Apache Software Foundation e sua documentação pode ser acessada na página oficial do projeto na web no endereço http://netbeans.apache.org.

XAMPP

O XAMPP é uma distribuição Apache totalmente gratuita e fácil de instalar, que contém MariaDB, PHP e Perl. O pacote de software livre XAMPP foi configurado para ser incrivelmente fácil de instalar e usar. A documentação oficial do XAMPP, atualmente em sua versão 7.3.12, pode ser acessada diretamente na web pelo endereço https://www.apachefriends.org/index.html.

Lúcio Lemos Postado em Dezembro 13, 2019

Muitos desenvolvedores em sua fase inicial de aprendizado sabem por experiência própria que não é fácil instalar um servidor web Apache integrado a MySQL, PHP e Perl. O objetivo do XAMPP é oferecer aos desenvolvedores de todos os níveis, uma maneira rápida e fácil de instalar e rodar se servidor web.

O Mint é executado no ambiente de desktop do Cinnamon 4.0 e vem com o linux-firmware 1.173.2 e o kernel do Linux 4.15.0-20. Inclui mudanças que tornam o login no desktop mais fácil e amigável. Utiliza por padrão o desktop Cinnamon, um derivado do GNOME, com um menu no painel inferior junto à barra de tarefas (o MintMenu), similar ao menu do KDE, ou ao menu "Iniciar" do Windows. É um sistema operacional moderno, elegante, confortável e fácil de usar. Possui suporte multimídia pronto para o uso, incluindo alguns softwares proprietários e uma ampla gama de aplicativos gratuitos e de código aberto.

PHP

O PHP, acrônimo recursivo para "Hypertext Preprocessor" é uma linguagem de script open source de uso geral, muito utilizada, e especialmente adequada para o desenvolvimento web e que pode ser embutida dentro do HTML.
A documentação oficial do Php, pode ser acessada diretamente na web pelo endereço https://www.php.net.

Lúcio Lemos Postado em Dezembro 13, 2019

O que distingue o PHP de algo como o JavaScript no lado do cliente é que o código é executado no servidor, gerando o HTML que é então enviado para o navegador. O navegador recebe os resultados da execução desse script, mas não sabe qual era o código fonte. Você pode inclusive configurar seu servidor web para processar todos os seus arquivos HTML com o PHP, e então não há como os usuários dizerem o que você tem na sua manga.

O PHP pode ser utilizado na maioria dos sistemas operacionais, incluindo Linux, várias variantes do Unix (como HP-UX, Solaris e OpenBSD), Microsoft Windows, Mac OS X, RISC OS e provavelmente outros. O PHP também tem suporte à maioria dos servidores web atualmente e isso inclui o Apache

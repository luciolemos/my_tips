O Linux Mint é uma distribuição, baseada no Ubuntu 18.04.1 LTS

Lúcio Lemos Postado em Dezembro 13, 2019

O Mint é executado no ambiente de desktop do Cinnamon 4.0 e vem com o linux-firmware 1.173.2 e o kernel do Linux 4.15.0-20. Inclui mudanças que tornam o login no desktop mais fácil e amigável. Possui duas versões: uma baseada em Ubuntu (com o qual é totalmente compatível e partilha os mesmos repositórios) e outra baseada em Debian. Suporta muitos idiomas, inclusive o português e utiliza o Cinnamon como seu principal ambiente de desktop. Diferencia-se do Ubuntu e do Debian por incluir drivers e codecs proprietários por default e por alguns recursos que permitem fazer em modo gráfico(GUI) configurações que em ambos os sistemas são feitas através de linha de comando (CLI). Utiliza por padrão o desktop Cinnamon, um derivado do GNOME, com um menu no painel inferior junto à barra de tarefas (o MintMenu), similar ao menu do KDE, ou ao menu "Iniciar" do Windows. É um sistema operacional moderno, elegante, confortável e fácil de usar. Possui suporte multimídia pronto para o uso, incluindo alguns softwares proprietários e uma ampla gama de aplicativos gratuitos e de código aberto.

1. Navegando entre diretórios a partir da raiz: luciolemos@dell:~$ cd /opt/lampp/htdocs

2. Criando um diretório com permissões de administrador: luciolemos@dell:/opt/lampp/htdocs$ sudo mkdir dicas

3. Verificando as permissões: luciolemos@dell:/opt/lampp/htdocs$ ls -lt

4. Dando ao usuário permissão de escrita no servidor lampp: luciolemos@dell:~$ sudo chmod -R 777 /opt/lampp/htdocs

XAMPP

Instalando o XAMPP:

my tips »

1. Baixando dos repositórios os pacotes:
luciolemos@dell:~$ wget "https://ufpr.dl.sourceforge.net/project/xampp/XAMPP%20Linux/7.3.10/xampp-linux-x64-7.3.10-0-installer.run" -O xampp-installer.run

2. Dando permissão de escria no doretório: luciolemos@dell:~$ chmod +x xampp-installer.run

3. Iniciando o instalador: luciolemos@dell:~$ sudo ./xampp-installer.run

4. Iniciando o gerenciador do XAMPP no modo gráfico: luciolemos@dell:~$ sudo /opt/lampp/manager-linux-x64.run

5. Para iniciar todos os serviços do XAMPP, use o seguinte comando no terminal: luciolemos@dell:~$ sudo /opt/lampp/lampp start

6. Para finalizar todos os serviços do XAMPP: luciolemos@dell:~$ sudo /opt/lampp/lampp stop

7. Reinicializando todod os serviços do XAMPP: luciolemos@dell:~$ sudo /opt/lampp/lampp restart

Ambiente gráfico é o nome dado à interface projetada com o objetivo de facilitar e tornar prática a utilização do computador por meio de representações visuais do sistema operacional como elementos gráficos, ícones e outros indicadores visuais.
Interface é por definição, o conjunto de recursos que permitem a um sistema adaptar-se a outro, seja ele físico ou virtual.

Do ponto de vista do usuário de TI, existem dois tipos de interfaces: GUI (Graphic User Interface) e CLI (Command Line Interface).

A GUI é composta por todos os recursos gráficos utilizados pelo usuário para que o mesmo consiga executar suas tarefas facilmente, sem a necessidade de conhecimento profundo de comandos. O cursor piscando intermitentemente em nossa tela, os ícones em nosso desktop dos aplicativos nativos ou instalados em nosso sistema, botões de acão, links e o próprio assistente de instalação de programas são alguns exemplos de elementos da Interface Gráfica do Usuário.

A CLI por sua vez é destinada aos usuários avançados do sistema e que necessitam executar tarefas não acessíveis, ou não disponíveis no modo GUI. O emulador de terminal para o Ambiente de Trabalho MATE no Linux Mint e o CMD do Windows são exemplos de interfaces que utilizam linha de comando.

O Linux Mint é um sistema operacional moderno, elegante e confortável, poderoso e fácil de usar․ É uma das distribuições Linux de desktop mais populares do mundo․

Lúcio Lemos Postado em Dezembro 13, 2019

Sua versão estável, o Linux Mint 19.2, codinome ″Tina″ é uma versão LTS (versão de suporte de longo prazo), sigla inglesa para Long Term Support, suportada até o ano de 2023. O Tina vem com software atualizado, traz aprimoramentos e muitos novos recursos que tornam sua experiência com a área de trabalho mais agradável․

Graças a uma abordagem conservadora de atualizações de software, um Update Manager exclusivo e a robustez de sua arquitetura, o Linux Mint requer pouca manutenção (sem regressões, antivírus, anti-spyware ... etc)․

Diferencia-se do Ubuntu e do Debian por incluir drivers e codecs proprietários por default e por alguns recursos que lhe permitem fazer em modo gráfico(GUI), configurações que em outros sistemas são feitas através de linha de comando (CLI). Utiliza por padrão o desktop Cinnamon, um derivado do GNOME, com um menu no painel inferior junto à barra de tarefas (o MintMenu), similar ao menu do KDE, ou ao menu ″Iniciar″ do Windows. Suas principais caracteŕistcas são:

  • Funciona imediatamente, com suporte completo a multimídia e é extremamente fácil de usar;
  • É gratuito e de código aberto;
  • É orientado pela comunidade;
  • Baseado no Debian e Ubuntu, fornece cerca de 30.000 pacotes e um dos melhores gerenciadores de software;
  • É seguro e confiável; e
  • Suporta muitos idiomas, inclusive o português e utiliza o Cinnamon como seu principal ambiente de desktop.
  • …

Linux Mint - Tips de comandos no terminal

  1. Navegando até o diretório htdocs a partir da raiz:
    luciolemos@dell:~$ cd /opt/lampp/htdocs
  2. Criando o diretório dicas dentro de htdocs:
    luciolemos@dell:/opt/lampp/htdocs$ sudo mkdir dicas
  3. Verificando as permissões para o diretório htdocs:
    luciolemos@dell:/opt/lampp/htdocs$ ls -lt
  4. Dando ao usuário luciolemos, permissão de escrita no diretório htdocs:
    luciolemos@dell:~$ sudo chmod -R 777 /opt/lampp/htdocs

O Apache NetBeans IDE é um Ambiente de Desenvolvimento Integrado - uma ferramenta para programadores, que permite escrever, compilar, depurar e instalar programas.

Lúcio Lemos Postado em Dezembro 13, 2019

O projeto foi criado em junho de 2000 pela Sun Microsystems. O IDE é completamente escrito em Java, mas pode suportar qualquer linguagem de programação. Existe ainda um grande número de módulos que permitem extender suas funcionalidades.

O Apache Netbeans IDE é um produto livre, sem restrições quanto à sua forma de utilização. Sua última versão, o Apache NetBeans 11.2, pode ser instalado em todos os sistemas operacionais compatíveis com Java, como Windows, Linux, Mac OSX e BSD. O Write Once, Run Anywhere, também se aplica ao NetBeans.

Com seu Editor Java em constante aprimoramento, muitas funcionalidades avançadas e uma extensa linha de ferramentas, modelos e exemplos, o NetBeans IDE define o padrão de desenvolvimento com suas tecnologias inovadoras. Algumas de suas característcas são:

  • É gratuito e de código aberto
  • Realça o código-fonte sintática e semânticamente;
  • Permite refatorar facilmente seu código;
  • Incorpora uma variedade de ferramentas úteis e poderosas;
  • O Editor do NetBeans recua linhas, associa palavras e colchetes;
  • Fornece modelos de código, dicas de codificação e ferramentas de refatoração;
  • Suporta várias linguagens, incluindo Java, C/C++, XML, HTML, PHP, Groovy, Javadoc, JavaScript e JSP;
  • Editor extensível, para você poder adicionar suporte para muitas outras linguagens; e
  • Integração inovadora com ferramentas de versionamento como o Subversion, o Mercurial e o Git;
  • Auto-completar avançado;
  • Total suporte ao ANT, ferramenta de automatização da construção de programas;
  • Suporte a Databases;
  • Suporte ao GitHub;
  • Editor integrado ao Emmet;
  • …

GitHub é uma plataforma de hospedagem de código-fonte com controle de versão usando o Git, criada para desenvolvedores. Ele permite que programadores, utilitários ou qualquer usuário cadastrado na plataforma contribuam em projetos privados e/ou Open Source de qualquer lugar do mundo․

Lúcio Lemos Postado em Dezembro 13, 2019

Agora que sabemos o que é o GitHub, precisamos entender que é o Git. O Git é um sistema de controle de versão desenvolvido por Linus Torvalds (o criador do Linux).

Sempre que desenvolvedores criam um novo projeto, são necessárias atualizações no código base, afim de permitir atualização de versões, correção de bugs, adição de novas ferramentas, etc. O sistema de controle de versão ajuda a acompanhar as mudanças feitas no código base. E mais, ele também registra quem efetuou a mudança e permite a restauração do código removido ou modificado. Em 4 de junho de 2018, a Microsoft anunciou que havia chegado a um acordo para adquirir o GitHub por US $ 7,5 bilhões. A compra foi encerrada em 26 de outubro de 2018.

O Emmet é um plug-in para muitos editores de texto populares, que melhora a produtividade e aprimora bastante o fluxo de trabalho de desenvolvedores HTML e CSS. É escrito em JavaScript puro e funciona em diferentes plataformas: navegador da web, Node.js, Microsoft WSH e Mozilla Rhino. Um kit de ferramentas essencial para desenvolvedores da web

Lúcio Lemos Postado em Dezembro 13, 2019

A maioria dos editores de texto disponíveis permite armazenar e reutilizar trechos de código que se repetem no código-fonte, chamados de ″snippets″. Embora os snippets sejam uma boa maneira de aumentar sua produtividade, todas as implementações têm armadilhas comuns: você precisa definir os snippets primeiro e não pode estendê-los em tempo de execução.

O Emmet leva a idéia de snippets a um nível totalmente novo: você pode digitar expressões semelhantes a CSS que podem ser analisadas dinamicamente e produzir saída dependendo do que você digitar na abreviação. O Emmet é desenvolvido e otimizado para desenvolvedores da Web cujo fluxo de trabalho depende de HTML / XML e CSS, mas também pode ser usado com linguagens de programação.

Abreviações são o coração do kit de ferramentas Emmet: essas expressões especiais são analisadas em tempo de execução e transformadas em bloco de código estruturado, HTML por exemplo. A sintaxe da abreviação se parece com seletores CSS com algumas extensões específicas para geração de código. Suas principais características são:

  • É gratuito e de código aberto
  • Sintaxe inspirada nos seletores CSS;
  • Escrito em javascript;
  • Codificação ultra-rápida;
  • …

O Apache é um servidor web de plataforma aberta e popular. É a principal tecnologia da Apache Software Foundation, responsável por mais de uma dezena de projetos envolvendo tecnologias de transmissão via web, processamento de dados e execução de aplicativos distribuídos. A versão mais recente do servidor é a 2.4.

Lúcio Lemos Postado em Dezembro 13, 2019

Criado em 1995 por Rob McCool, na época funcionário da NCSA (National Center for Supercomputing Applications), o servidor Apache ou Servidor HTTP Apache é o mais bem sucedido servidor web livre que existe. Trata-se de um servidor muito popular e utilizado principalmente com Sistemas Operacionais Linux.

Conforme anunciado, o Apache HTTP Server Project interrompeu todo o desenvolvimento e a revisão de patches da série 2.2.x de lançamentos, que foi descontinuado a partir do lançamento da nova versão em 14 de agosto de 2019.

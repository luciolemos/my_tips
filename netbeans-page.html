Apache Netbeans

Ambiente de Desenvolvimento Integrado, Plataforma de Ferramentas e Estrutura de Aplicação.

Lúcio Lemos Postado em Dezembro 13, 2019

O Apache NetBeans é muito mais que um editor de texto. Destaca o código-fonte de forma sintática e semânticamente, permite refatorar facilmente o código, com uma variedade de ferramentas úteis e poderosas. Desenvolva rápida e facilmente aplicativos de desktop, dispositivos móveis e Web com Java, JavaScript, HTML5, PHP, C / C ++ e muito mais. O Apache NetBeans é gratuito e de código aberto. É mantido pela Apache Software Foundation e atualmente encontra-se na versão 11.2

1. Navegando entre diretórios a partir da raiz: luciolemos@dell:~$ cd /opt/lampp/htdocs

2. Criando um diretório com permissões de administrador: luciolemos@dell:/opt/lampp/htdocs$ sudo mkdir dicas

3. Verificando as permissões: luciolemos@dell:/opt/lampp/htdocs$ ls -lt

4. Dando ao usuário permissão de escrita no servidor lampp: luciolemos@dell:~$ sudo chmod -R 777 /opt/lampp/htdocs

Ambiente de Desenvolvimento Integrado - IDE

Um Ambiente de Desenvolvimento Integrado, do inglês Integrated Development Environment (IDE) é um programa de computador que reúne características e ferramentas de apoio ao desenvolvimento de software.

Lúcio Lemos Postado em Dezembro 13, 2019

O trabalho de um desenvolvedor é complexo e demorado. Ter em mãos ferramentas de trabalho sofisticadas, torna o trabalho mais rápido, fácil e produtivo. Por essa razão, a maioria dos desenvolvedores recorrem a ambientes de desenvolvimento integrados para gerenciar o fluxo de trabalho, acessar um conjunto de ferramentas de desenvolvimento em um único painel e reduzir erros que podem ter consequências catastróficas se implantados sem verificação. O uso de IDE, facilita a técnica de RAD (Rapid Application Development), ou Desenvolvimento Rápido de Aplicativos, que visa a maior produtividade dos desenvolvedores.
